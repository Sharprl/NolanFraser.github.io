# Nolan Fraser Cybersecurity Portfolio

Simple multi-page portfolio website for GitHub Pages.

## Pages

- `index.html` - Home
- `about.html` - About me
- `skills.html` - Cybersecurity skills
- `projects.html` - Project highlights
- `resume.html` - Embedded and downloadable resume
- `contact.html` - Contact information

## Resume Setup

1. Put your real resume PDF at:
   - `assets/resume.pdf`
2. Keep the filename exactly `resume.pdf` so all links continue to work.

## Publish With GitHub Pages

1. Create a GitHub repository and upload all files in this folder.
2. In GitHub, open **Settings -> Pages**.
3. Under **Build and deployment**, set:
   - **Source:** Deploy from a branch
   - **Branch:** `main`
   - **Folder:** `/ (root)`
4. Save and wait for GitHub Pages to publish.
5. Open the URL GitHub shows in Pages settings to verify.

## Notes

- This site uses only static HTML/CSS/JS, so no build step is needed.
- Before publishing, replace the temporary LinkedIn/email text on `contact.html`.
